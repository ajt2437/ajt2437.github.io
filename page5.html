Important note: The class name is case sensitive.

HTML elements can have more than one class name, each class name must be separated by a space.

The class name can also be used by JS to perform certain tasks for elements with the specified class name. JS can access elements with a specified class name by using the getElementsByClassName() method.

Special note: The id value is case-sensitive

Difference between Class and ID

An HTML element can only have one unique id that belongs to that single element, while a class name can be used by multiple elements.

Using the id Attribute in JS

You can access an element in JS with a specified id by using the getElementById() method.

A browser without support for JS will show the text written inside the noscript element.

HTML File paths

Path Description
<img src="picture.jpg"> picture.jpg is located in the same folder as the current page
<img src="images/picture.jpg"> pictures.jpg is located in the images folder in the current folder.
<img src="/images/picture.jpg"> pictures.jpg is located in the images folder at the root of the current web.
<img src="../picture.jpg"> pictures.jpg is located in the folder one level from the current folder.

Best practice

It is best practice to use relative file paths (if possible)
