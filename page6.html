The HTML <head> Element

It is a container for metadata (data about data)

The following tags describe metadata:

  • <title>
    • Required in all HTML/XHTML documents
    • Provides a title when added to favourites.
    • Displays a title for the page in search engine results
  • <style>
  • <meta>
    • Used to specify which charset is used
    • Page description
    • Keywords
    • Author
    • And other metadata.
  • <link>
    • Used to link to external style sheets
  • <script>
  • <base>

Setting The Viewport

You should include the following <meta> viewport element in all your webpages:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

The HTML <base> Element

The <base> element specifies the base URL and base target for all relative URLs in a page:

<base href="https://ajt2437.github.io/" target="_blank">

Special notes:Its not recommended omitting the <html> and <body> tags. Omitting these tags can crash DOM or XML software and produce errors in older browsers (IE9). However, omitting the <head> tag has been a common practice for quite some time now.
